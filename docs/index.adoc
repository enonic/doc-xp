== Apps

Standalone software packages that can be installed and started on Enonic XP with access to all platform services and APIs. Use them to add specific features and functionality to your instance, ranging from APIs to administrative interfaces.

TIP: *Building your own apps?*  Visit https://developer.enonic.com/docs/code[Enonic Development Kit] documentation. 

TIP: *Visit https://market.enonic.com[Enonic Market]* for official apps from Enonic, and 3rd party developers.


== Platform

The platform provides essential services and APIs that applications leverage to build their functionality.

* <<endpoints#,Gateway>> - HTTP entry point for 3rd party client interactions.
* <<apps#,App runtime>> - Able to power and run multiple applications simultaneously.
* <<security/iam#,Identity and Access Management (IAM)>> - A comprehensive security model for authentication and authorization across the platform.
* Clustering - for distributed memory caching, events and job execution.
* <<storage#,NoSQL storage>> - A built-in NoSQL storage engine for storing content, application- and system data.
