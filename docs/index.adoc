== About

Enonic XP is an API first CMS and back-end for your digital experiences.

* Augmentable Headless Content API
* Schema system for structured content
* Advanced search and querying
* Extend with JavaScript/TypeScript
* Flexible and pluggable end-to-end IAM
* Portable and Open Source
