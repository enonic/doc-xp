== More than an API
Enonic XP aims to be the ultimate CMS and back-end for your digital experiences. It provides a powerful and flexible foundation where you can customize and extend every aspect of the platform to fit your unique needs - without starting from scratch.

In order to provide this flexibility, XP was built like an operating system, with a core platform that provides essential services and APIs, and an ecosystem of applications that can be installed on top to add specific functionality. 

This architecture allows you to pick and choose the capabilities you need, while still having the option to build custom applications or extend existing ones. The platform provides the building blocks, and you can assemble them in whatever way makes sense for your project.
